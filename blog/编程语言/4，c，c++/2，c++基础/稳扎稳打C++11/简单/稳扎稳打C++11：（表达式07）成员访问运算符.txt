成员访问运算符有两个：点运算符（dot operator）和箭头运算符（arrow operator）。
点运算符从一个类类型的对象获取成员。
箭头运算符ptr->mem是(*ptr).mem的同义词。
箭头操作符需要一个指针操作数并返回一个左值。
如果对象是左值，那么点运算符返回的是左值，否则是右值。

#include <iostream>
#include <vector>
 
using namespace std;
 
int main()
{
    vector<int> iv = {1,3,5,7,9};
    vector<int> *piv = &iv;
    cout << iv.size() << endl;
    cout << piv->size() << endl;
     
    /*
    Environment:
        g++4.8.1(32bit)
    Output:
        5
        5
    */
     
    return 0;
}
