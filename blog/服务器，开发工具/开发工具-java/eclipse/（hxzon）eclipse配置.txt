2.4，java语法配色
java -》 editor -》 syntax coloring

背景色：黑色

次要：

注解：-1，-3
关键字（除return外）：3,1，粗体，斜体
括号：-1，-3

重要：

类名，方法调用（非自身或父类）：1，-1（粉红）
泛型参数
方法定义：1，-1

静态变量，静态常量：1,1（橙色），粗体，斜体

方法参数vs本地变量vs全局变量vs继承的全局变量
方法参数：白色
本地变量：白色，定义处粗体
全局变量：白色，粗体，下划线，斜体
继承的全局变量：白色，斜体

自身方法调用vs父类方法调用
自身方法调用（methods）：3,2
父类方法调用：3,2，下划线，斜体

字符串：1,5
return语句：3,2，下划线，斜体
